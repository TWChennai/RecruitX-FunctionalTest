# Functional Tests

RecruitX hybrid app functional tests Framework.

###Setup Automation Framework
- Install node v5.9.1
- Install Appium 1.3.7 GUI server https://github.com/appium/appium-dot-app
- npm Install

###Steps to run Android tests
- VERSION='4.4' npm run android
